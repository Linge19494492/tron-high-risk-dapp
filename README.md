# Tron high risk DAPP
An open-source Highrisk DAPP (Decentralized application) on Tron blockchain


## Smart contract Compile and Deployment
Compile and deployment can be achieved via [Tronscan](https://www.tronscan.org).

## Prerequisites
```
npm install
```


## Start DAPP
```
npm run dev
```
